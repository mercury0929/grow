# GROW — 筋トレの成長を見える化

筋トレの記録・成長可視化を行う、**単一HTMLファイル**のWebアプリです。
サーバー不要・登録不要・データは端末内 (localStorage) に保存されるため、プライバシーも安心。

ガチ勢向けに「セット毎の重量・レップ記録」を中心に設計しています。

## 主な機能

- **セット毎の記録** — 1セット目 60kg×10、2セット目 65kg×8… のように、セットごとに異なる重量・レップを記録
- **種目管理** — カテゴリ (胸・背中・脚・肩・腕・体幹) 別。プリセット種目あり、追加・リネーム可能。自重種目・時間種目 (プランク等) にも対応
- **前回の記録 / メモを自動表示** — 種目を選ぶと前回構成が自動入力、前回メモも表示
- **自己ベスト (推定1RM)** — Epley式で算出、更新時にお祝い表示
- **目標設定** — 種目ごとに目標1RMを設定、進捗バー + 達成時のお祝い
- **週次ストリーク** — 週N回の目標を達成した連続週数を表示
- **分析** — トレーニングヒートマップ / 週ボリューム / バランス診断 (前後・上下・大小) / 成長ランキング / 種目別推移
- **体重記録** — 推移を折れ線グラフで表示
- **絞り込み** — 種目別 / メモありフィルタで履歴を遡れる
- **選択削除 + Undo** — 複数選択で一括削除、5秒間の取り消し可能
- **プレート計算機** — 目標重量からバー+片側プレート構成を計算
- **セット間タイマー** — 60/90/120/180秒 + カスタム
- **共有** — 期間指定のサマリーをPDFで出力 (個人情報マスク選択あり)
- **バックアップ** — JSONエクスポート/インポートでデータ移行・復元、CSV出力
- **ライト / ダークテーマ** — システム連動

## 使い方

### そのまま使う
`index.html` をブラウザで開くだけ。インターネット接続はグラフ描画ライブラリ (CDN) の読み込みにのみ必要です。

### Web公開版 (GitHub Pages)
👉 **https://mercury0929.github.io/grow/**

スマートフォンでアクセスし「ホーム画面に追加」すると、ネイティブアプリのように使えます (PWA対応)。

## 技術構成

- HTML + CSS + JavaScript の単一ファイル、ビルド工程なし
- [Chart.js](https://www.chartjs.org/) — グラフ描画 (CDN)
- [html2canvas](https://html2canvas.hertzen.com/) + [jsPDF](https://github.com/parallax/jsPDF) — PDF出力 (CDN)
- データ保存: ブラウザの localStorage

## データについて

すべての記録はお使いのブラウザ内 (localStorage) にのみ保存されます。サーバーには一切送信されません。
ブラウザのデータ削除や端末変更で消えるため、定期的に「移行用エクスポート (JSON)」でバックアップすることを推奨します。
